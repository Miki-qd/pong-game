# Pong Game Documentation

## Introduction
This Pong Game is a classic arcade game where two players control paddles to hit a ball back and forth.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Miki-qd/pong-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pong-game
   ```
3. Install dependencies if any.

## Usage
Run the game using:
```bash
python pong_game.py
```

## Controls
- Player 1: Use `W` and `S` to move up and down.
- Player 2: Use the `Up Arrow` and `Down Arrow` keys to move.
